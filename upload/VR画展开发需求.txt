一、产品概述
1.1 产品定位
为校外教育培训机构（美术班）打造的VR线上画展平台。每个学生拥有一个独立的虚拟展厅，墙面挂满自己的画作，就像拥有了一场属于自己的画展。

1.2 核心价值
对谁	价值
学生	拥有属于自己的画展，成就感满满
家长	随时随地通过手机观看孩子的作品，感受孩子的成长
机构	提升品牌形象，展示教学成果，增强家长信任
1.3 三条铁律（最高优先级）
🏆 必须高端 — 家长一看就觉得专业、有品质
⚡ 必须流畅 — 不能卡顿，家长没有耐心等待
🔗 必须好分享 — 微信一键分享，家长点开即看
二、用户角色
2.1 角色定义
角色	描述	是否需登录	核心操作
管理员（老师）	机构工作人员	暂不需要	管理学生、上传画作、选风格、生成链接
学生	美术班学员	暂不需要	查看自己的展厅
家长	学生家长	暂不需要	通过分享链接观看展厅
📌 V1.0 不实现登录系统，所有页面公开访问。后续版本迭代时再加。

三、产品功能

分享	每个学生卡片有独立的分享按钮（生成微信分享链接）

### 3.2 3D展厅（核心体验）

**功能描述**: 360°全景3D虚拟画廊，沉浸式浏览学生画作

#### 3.2.1 展厅空间

| 要素 | 说明 |
|------|------|
| 空间形态 | 一个独立房间，四面墙挂画 |
| 画作数量 | 5-10幅（V1.0） |
| 天花板 | 有天花板，营造封闭空间感 |
| 地板 | 根据风格不同（木地板/地毯等） |
| 灯光 | 聚光灯照射每幅画，环境光柔和 |

#### 3.2.2 展厅风格（V1.0 提供两种）

**风格A — 现代简约白墙画廊**

白色/浅灰墙壁
浅色木地板
顶部射灯聚焦每幅画
简约画框（黑色/金色细框）
干净、专业、像真正的美术馆。
**风格B — 温馨家居风**
暖色墙壁（米白/浅黄）
木质地板/地毯
沙发、茶几、书架等家具
装饰画框（木质/复古框）
温馨、舒适、像家里的客厅

> 📌 V1.0 由管理员统一选择一种风格。后续可支持每个学生不同风格。

#### 3.2.3 画作展示

| 要素 | 说明 |
|------|------|
| 画框 | 有真实质感的画框（根据风格不同） |
| 画旁文字 | **V1.0 不显示文字**，画面本身即可 |
| 聚光灯 | 每幅画上方有独立聚光灯 |
| 画作间距 | 合理间距，不拥挤 |
| 点击查看 | 点击/触摸画作 → 弹出高清大图 + 画作信息浮层 |

#### 3.2.4 画作详情浮层

点击画作后弹出：
- 画作高清大图（支持缩放）
- 画作名称
- 创作时间（如有）
- 简短描述（如有）
- 关闭按钮

#### 3.2.5 学生信息

在展厅入口处展示（可选填）：
- 学生姓名
- 年龄/班级
- 简短介绍（如"小明，8岁，学习画画2年"）

#### 3.2.6 交互方式

**PC端**:
| 操作 | 效果 |
|------|------|
| 鼠标移动 | 转动视角（360°） |
| WASD / 方向键 | 前后左右移动 |
| 点击画作 | 弹出详情 |
| ESC | 释放鼠标 |

**手机端**:
| 操作 | 效果 |
|------|------|
| 手指滑动 | 转动视角 |
| 双指缩放 | 放大/缩小 |
| 双击/长按画作 | 弹出详情 |
| 虚拟摇杆（可选） | 前后左右移动 |

### 3.3 管理后台

**功能描述**: 管理员操作的后台界面

#### 3.3.1 学生管理

| 功能 | 说明 |
|------|------|
| 添加学生 | 填写姓名、年龄/班级、简介（可选） |
| 编辑学生 | 修改学生信息 |
| 删除学生 | 删除学生及其展厅 |
| 学生列表 | 查看所有学生 |

#### 3.3.2 画作管理

| 功能 | 说明 |
|------|------|
| 上传画作 | 为指定学生上传图片（支持批量上传） |
| 替换画作 | 替换已上传的画作 |
| 删除画作 | 删除指定画作 |
| 排序 | 调整画作在墙上的展示顺序 |
| 预设画作 | 创建展厅时自动放置占位示例图 |

#### 3.3.3 展厅设置

| 功能 | 说明 |
|------|------|
| 选择风格 | 白墙画廊 / 温馨家居 |
| 生成分享链接 | 为每个学生生成独立的微信分享链接 |

## 四、微信分享

### 4.1 分享卡片

微信分享时显示：
- **标题**: "[学生姓名]的线上画展"
- **描述**: "[机构名称] · 学生作品展示"
- **缩略图**: 该学生的代表画作

### 4.2 分享链接

- 每个学生有独立的专属链接
- 家长点开链接直接进入该学生的3D展厅
- **无需登录**，打开即看

### 4.3 兼容性

- 优先保证**微信内置浏览器**能流畅运行
- 如微信内置浏览器性能不足，引导到外部浏览器打开

## 五、技术要点

### 5.1 性能要求

| 指标 | 要求 |
|------|------|
| 首屏加载 | ≤ 3秒 |
| 展厅加载 | ≤ 5秒 |
| 帧率 | ≥ 30fps（手机端） |
| 图片加载 | 渐进式加载，先模糊后清晰 |

### 5.2 兼容性

| 平台 | 要求 |
|------|------|
| 微信内置浏览器 | ✅ 必须支持 |
| 手机 Safari | ✅ 必须支持 |
| 手机 Chrome | ✅ 必须支持 |
| PC Chrome/Edge | ✅ 必须支持 |
| PC Safari | ✅ 应该支持 |

### 5.3 规模

| 维度 | 数量 |
|------|------|
| 学生/展厅数 | 10-50个 |
| 每个展厅画作 | 5-10幅 |
| 单张图片大小 | 1-3MB |

### 5.4 技术方案建议

| 模块 | 方案 |
|------|------|
| 3D渲染 | Three.js + React Three Fiber（360°全景场景） |
| 前端框架 | Next.js 16 + TypeScript |
| UI组件 | Tailwind CSS + shadcn/ui |
| 数据库 | SQLite（Prisma ORM） |
| 图片存储 | 本地 /public 目录 |
| 状态管理 | Zustand |
